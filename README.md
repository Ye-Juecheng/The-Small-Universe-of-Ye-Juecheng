# The-Small-Universe-of-Ye-Juecheng
🌌 叶珏成的小宇宙

      “他要送给程心一颗星星”
      ——《三体》刘慈欣

一个基于 Three.js 打造的沉浸式 3D 星系网页，还原《三体》中宇宙级的浪漫，每个人都能在星海中注册、点亮、分享属于自己的恒星，构建独一无二的私人宇宙。
🌟 项目灵感
以星云、旋臂、恒星为视觉基底，让每一位用户都能成为恒星的拥有者，在虚拟宇宙中留下专属坐标，将文字里的浪漫变为可交互的视觉体验。
✨ 核心功能
3D 动态星系：螺旋星系实时渲染，恒星脉动发光，支持拖拽旋转、滚轮缩放
恒星注册：自定义恒星名称、拥有者、星座、质量等级，自动生成唯一星 ID
星座系统：10 大经典星座配色区分，视觉辨识度拉满
数据持久化：LocalStorage 本地存储，刷新不丢失星星数据
星星详情：点击恒星查看完整信息（坐标、注册时间、等级等）
数据导出 / 导入：一键导出 JSON 分享星系，加载他人星星数据共建宇宙
响应式设计：完美适配桌面 / 移动端，随时随地探索星海
视角重置：一键回到初始视角，操作极简
🛠 技术栈
3D 渲染：Three.js + OrbitControls
页面结构：HTML5 + CSS3
交互逻辑：原生 JavaScript (ES Module)
数据存储：LocalStorage
视觉效果：Canvas 粒子系统、Shader 着色器、渐变发光动画
📖 使用指南
启动动画：等待 7 秒开场文案，自动进入星系主界面
注册恒星：点击「注册你的星星」，填写信息点亮专属恒星
查看星海：点击「查看所有星星」，快速定位并聚焦恒星
导出数据：「导出星星数据」生成 JSON，分享给好友
加载数据：选择他人分享的 JSON，加载更多恒星到你的星系
视角控制：拖拽旋转 | 滚轮缩放 | 重置视角

💫 每一颗星星，都是宇宙写给你的情书
